# Evidence Register

## Purpose

This register serves as the authoritative catalog of evidence contained within the Gildmark portfolio.

Each entry represents a concrete artifact produced during the conception, design, implementation, or governance of the platform. Rather than cataloging capabilities directly, the register catalogs the evidence from which those capabilities can be legitimately inferred.

Every significant claim made within this portfolio—including résumé statements, interview talking points, architectural essays, and technical narratives—should trace back to one or more artifacts documented here.

The objective is to ensure that the portfolio remains evidence-driven rather than assertion-driven.

---

## Evidence Classification

| Strength        | Definition                                                                                                                      |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| **Exceptional** | Demonstrates capability rarely documented or executed at an individual contributor level. Represents a distinguishing strength. |
| **Strong**      | Clearly demonstrates professional capability beyond routine implementation.                                                     |
| **Supporting**  | Reinforces capabilities demonstrated elsewhere.                                                                                 |
| **Planned**     | Intended portfolio artifact not yet produced.                                                                                   |
